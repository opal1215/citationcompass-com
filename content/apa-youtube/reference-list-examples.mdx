# Reference list examples

<AnswerCard title="TL;DR" tldr="Reference list entries for YouTube videos include the author or channel, the upload date, the title in sentence case, the medium [Video], the platform (YouTube) and the URL." />

The full citation in your reference list contains all the information
readers need to locate the video.  Format the entry as follows:

**Author. (Year, Month Day). Title [Video]. YouTube. URL**

Write the title in **sentence case** (only the first word and proper
nouns are capitalised) and place "[Video]" in square brackets to
identify the medium.  End the entry with the platform name and a
link to the video.

<DataBox title="Example reference entries">
  <p>CrashCourse. (2015). The French Revolution [Video]. YouTube. https://www.youtube.com/watch?v=lTTvKwCylFY</p>
  <p>Khan Academy. (2020, May 5). Photosynthesis [Video]. YouTube. https://www.youtube.com/watch?v=Hb2-GXeW19g</p>
  <p>NASA. (n.d.). Perseverance rover landing [Video]. YouTube. https://youtu.be/0qiKvVooTsM</p>
</DataBox>

If no date is available, use **(n.d.)** in place of the date.  When
the author is unknown, substitute the channel name.  For archived
live streams, cite the date on which the stream aired.
