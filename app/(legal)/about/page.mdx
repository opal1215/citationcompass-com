# About

Welcome to **APA × YouTube Citation Generator & Guide**.  This project
provides a free, educational resource to help students, researchers
and creators cite YouTube videos in accordance with the APA 7th
edition guidelines.  It is built using open source technologies and
offers a citation generator along with a collection of how‑to
articles covering special cases.

If you have questions or feedback, please visit our [Contact](/contact)
page.
